## Разворачивание проекта

Создаем файл с переменными окружения из файла-примера

```
cp .env.example .env

```
В файле .env в переменную вводим абсолютный путь к файлу

````
FILEPATH=/путь/к/файлу/text.txt (пример на картинке)
````
